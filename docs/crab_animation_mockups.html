🦀 ClaudeBar — Crab walking + usage + reset

ปูเดินเป็นพระเอก (ขาสลับ 2 เฟรม + ตัวเด้ง + ก้ามโบก). โจทย์คือผูก 3 อย่าง: ปู(animation) · usage(รูป) · reset(เวลา). ด้านล่างคือ 4 แนวทางจัดวาง — ซ้าย = ขนาดจริงบน menu bar, ขวา = ซูม.

1 · ปูเดินบน "ชายหาด progress" — usage = ระยะที่เดินได้ (แนะนำ)

พื้นทราย/น้ำเป็นแถบ progress: ส่วนที่ถมแล้ว = usage%. ปูยืน "เดินอยู่กับที่" ตรงหัวแถบที่ถม (ตำแหน่ง = usage จริง ไม่หลอกตา). เวลารีเซ็ตอยู่ที่ title. ครบ 3 อย่างในไอคอนเดียว + กว้างพอให้ปูชัด.
